# PNG 图例 + 色号点击高亮

## Goal

让导出 PNG 和网页预览更适合「对着图数豆 / 按色号摆豆」：

1. 导出 PNG 在图纸下方附带本图实际用色统计（色块 + `code:count`，按用量降序）。
2. 网页点击色号后，画布轻微高亮该色所有格子，方便对照点数。

## Background

当前（代码已确认）：

- `drawPattern` / `exportPattern`（`src/lib/pattern.ts`）只画格子 + 可选网格/色号，无底部图例。
- 右侧 `colorUsage` 已按用量降序；整行点击 = 切换禁用，与高亮手势冲突。
- 预览 canvas 无 `highlightCode` 绘制参数。

用户价值：网页按色数豆；导出/打印时底部直接看到用色清单。

## Requirements

### R1：导出 PNG 底部用色统计（必选）

- **R1.1** 导出图片在图案区域下方增加图例区；图例为导出交付物的固定部分，不可省略。
- **R1.2** 仅列出 `pattern.counts` 中 `count > 0` 的色号；格式 `code:count`；按 count 降序，同 count 可按 code 排序。
- **R1.3** 布局：每项 = 小色块 swatch + `code:count` 文字；条目自动换行；画布高度随行数增加，不得裁切。
- **R1.4** 含简短标题「用色统计」，并展示用色种数与总豆数（如 `12 色 · 共 1600 豆`）。
- **R1.5** 图例与导出不受网页 `highlightCode` 影响；导出图案区不应用高亮淡化/描边。

### R2：网页色号点击高亮

- **R2.1** 状态 `highlightCode: string | null`；单击色卡行主体 → 设为该 code；再点同一 code → 清除；点其他 code → 切换。
- **R2.2** 绘制：有高亮时，非聚焦色整体降低透明度；聚焦色不透明 + 细描边；色号文字仍可读。
- **R2.3** 高亮仅影响预览绘制，不改 `pattern` 数据；导出不应用高亮。
- **R2.4** 仅 `count > 0` 且未禁用的色号可作为高亮目标；`count === 0` 单击不进入高亮（可无操作或轻提示，实现选最简：忽略）。
- **R2.5** 若当前高亮色被禁用或重新生成后该色 count 变为 0，自动清除 `highlightCode`。

### R3：禁用与高亮手势分离

- **R3.1** 色卡行 **单击主体 = 高亮**（方案 A）。
- **R3.2** **禁用 = 行尾独立图标按钮**（方案 B）；`title`/`aria-label` 标明「禁用此色」/「启用此色」；点击按钮 `stopPropagation`，不触发高亮。
- **R3.3** 禁用态视觉保持现有 disabled 样式；禁用行主体不可高亮。

## Acceptance Criteria

- [ ] AC1：导出 PNG 图案下方有「用色统计」图例，含 swatch + `code:count`，按 count 降序，且含种数与总豆数。
- [ ] AC2：用色很多时图例完整出现在 PNG 中（换行 + 增高画布，无裁切）。
- [ ] AC3：导出 PNG 图案区不出现高亮淡化/描边；图例始终为完整用色列表。
- [ ] AC4：网页单击某 `count > 0` 色号行主体后，该色格子可辨识高亮，其它色略淡，色号仍可读。
- [ ] AC5：再点同一色取消高亮；点另一色切换；点行尾禁用图标只切换禁用、不切换高亮。
- [ ] AC6：高亮色被禁用或该色用量归零后，高亮自动清除。
- [ ] AC7：`npm run build` 通过；现有网格/色号开关/PNG 下载流程不破坏。

## Constraints

- 纯前端 Canvas；无新运行时依赖。
- UI 文案中文。
- 不改色卡匹配 / median-cut / 调参管线语义。
- 保持 PWA 与本地图片处理承诺。

## Out of Scope

- PDF / 分板打印
- 多色同时高亮
- 制作模式进度标记
- 导出图例里的交互（PNG 为静态图）
- 色卡算法与数据结构重构

## Decisions

| # | 决策 | 选择 |
|---|------|------|
| 1 | 高亮 vs 禁用手势 | 单击行 = 高亮；禁用独立控件 |
| 2 | 导出图例 | 必选，底部统计 |
| 3 | 图例布局 | swatch + `code:count`，多行 wrap |
| 4 | 高亮视觉 | 非聚焦降透明 + 聚焦细描边；导出无高亮 |
| 5 | 禁用控件 | 行尾图标按钮 + 无障碍文案 |
